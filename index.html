passo a passo para executar as primeiras atualizações no Git

    Lista de comandos para executar o commit

  1. versionar pasta para repositorio git = git init
  2. criar arquivo = touch index.html
  3. verificar status = git status
  4. adicionar arquivo dentro = git add index.html
  5. fazer o commit e inserir a msg do que foi feito entre "" = git commit -m "fiz tais coisas nesse commit"
  6. mostar o que foi feito = git diff

    comandos utilizados

  • ls = mostra o que tem dentro da pasta/direitório
  • mkdir (nome) = cria pasta (nome)
  • cd nome = abre pasta nome
  • cd .. = volta uma pasta
  • code . = abre o vs code na pagina atual
  • code .. = abre vs code na página anterior
  • code (nome arq) = abre o arq (nome)
  • touch nome.formato = cria arquivo
  • rm (nome arq) = remove arq
  • ls - la = revela arq ocultos
  • git dif = mostra alterações realizadas

    como enviar do git para github

  1. abrir github
  2. criar novo repositório
  3. copiar url do repositório
  4. abrir git e digitar: git remote add origin + a url copiada do github
  5. digitar: git branch -m main
  6. digitar push -u origin main
